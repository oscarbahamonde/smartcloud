Las tecnologias que utilizamos para desarrollar esta API Rest son:

Python

Python es un lenguaje de programación interpretado, dialecto del estándar de bajo nivel denominado "Python 2" y lanzado en febrero de 1991 como parte de la plataforma de software "Python".

FastAPI

FastAPI es un framework de desarrollo de APIs RESTful que se basa en el modelo de desarrollo de microservicios.

NuxtJS

Nuxt.js es un framework de desarrollo de aplicaciones web moderno, de código abierto y de código fuente abierto.
